1. main.py:
   - Этот файл является точкой входа в приложение.
   - Здесь можно создать объект модели и вызывать методы для обуч
ения, оценки и использования модели.

2. model.py (ООП):
   - В этом файле определен класс модели, который инкапсулирует все
 операции, связанные с моделью машинного обучения.
   - Методы класса могут включать создание модели, компиляцию, обуч
ение, оценку и сохранение/загрузку модели.
   - Класс модели может иметь атрибуты, представляющие гиперп
араметры модели.

3. config.py:
   - В этом файле можно определить параметры модели, такие как гип
ерпараметры, пути к данным и другие конфигурационные настройки.
   - Параметры могут быть представлены в виде переменных или сл
оварей.

4. data.py:
   - В этом файле можно реализовать функции или классы для загрузки
 и предобработки данных.

5. train.py:
   - Этот файл содержит функции или методы, отвечающие за процесс
 обучения модели.
   - Здесь можно использовать данные из data.py для обучения модели и сохранения обученной модели.
